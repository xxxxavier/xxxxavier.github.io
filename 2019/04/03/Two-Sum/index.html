1. Two_Sum

4月 03, 2019

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example:

Given nums = [2, 7, 11, 15], target = 9,

Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

第一题，不是什么难题，大概一年多前刚开始刷leetcode的时候想的还是暴力破解$n^2$的方法，现在会马上想到使用map。我们可以用一个map把之前的值保存在遍历某个数的时候查找target-sum时候存在在map中，如果存在则是我们要的答案，这个方法的时间复杂度是O(n)。

从这道题延申的一点是，java和c++中map的实现，c++中，map, set, multimap, multiset都是通过红黑树实现，插入、查看、删除的时间复杂度都是O(logN), 而在java中是一种数组+链表的实现方式，先依靠hash得到key在数组中的大体位置，然后用链表来规避冲突，因此java中map查找的时间复杂度近似为O(1)。
