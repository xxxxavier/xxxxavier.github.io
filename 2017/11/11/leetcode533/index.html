Leetcode 533. Lonely Pixel II

11月 11, 2017

今天测试一下代码功能，正好今天刷了一道题Leetcode 533。

先贴一下题目

Given a picture consisting of black and white pixels, and a positive integer N, find the number of black pixels located at some specific row R and column C that align with all the following rules:

1.Row R and column C both contain exactly N black pixels.
2.For all rows that have a black pixel at column C, they should be exactly the same as row R

The picture is represented by a 2D char array consisting of ‘B’ and ‘W’, which means black and white pixels respectively.

这道题目读起来有些费劲，首先rule 1比较容易理解，x[R][C]为B时，第R行，第C列中是否都又N个black pixels。rule 2花了一段时间去理解，意思是要求，第C列中，所有的为’B’的行都要相同，也就是加入第C列为[‘B’,’B’,’W’,’B’,’W’],则第0，1，3行要完全相同。最初的理解是数量相同即可，第一次条之后发现是顺序必须相同，及，第0行为[‘W’,’B’,’B’,’B’],第1，3行也需要是[‘W’,’B’,’B’,’B’]。
